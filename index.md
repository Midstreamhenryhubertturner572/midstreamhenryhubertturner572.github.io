## 🔄 Setting up your sync tasks

The first time you open the software, the main screen appears. This screen allows you to manage your folders.

1. Click the Job button in the top left corner.
2. Select New to create a sync task.
3. Name your task based on the folder you want to back up.
4. Choose the type of task you need. A Sync task keeps two folders identical. A Backup task sends files from your computer to another location.
5. Click the side arrows to select the source folder on your computer.
6. Click the other side arrow to select your destination. This could be a folder on a thumb drive, another connected hard drive, or a cloud service like Google Drive or OneDrive.
7. Click the Analyze button to see what files the program intends to move.
8. Review the list and click Sync to start the data transfer.

## 🔍 Troubleshooting common issues

If you face difficulties, try these steps first:

- Application not opening: Restart your computer and try to launch the program again.
- Sync failing: Check your internet connection if you sync to a cloud service. Ensure you have enough storage space on the destination drive.
- File access denied: Right-click the folder and check your properties to ensure you have permission to read and write to that folder.
- Slow sync speeds: Close other programs that use the internet while you sync large amounts of data. Use a wired ethernet connection instead of Wi-Fi for better stability.
